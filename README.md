# CarManager
- ### C# WinForm json, mssql, xml을 활용한 주차 관리 프로그램
- ### 개발 환경 : VisualStudio 2019, DBMS : Microsoft SQL Server2012
- ### 기능 : 주차, 출차, 주차 공간
